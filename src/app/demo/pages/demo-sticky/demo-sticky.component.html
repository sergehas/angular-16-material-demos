This demo illustrate usage of the appStick

Overview

In a complex layout like in a angular material app, it is very hard to get the scrolling on the proper (content) area. Generally speaking, and as it is in this demo app, the scrolling area is the mat drawer (in app.component.html), not the "page" component (and event more, not a tab in a page component). This lead to the position:sticky CSS attribute is not working in a page component, becaus teh scrolling is applied to (grand) grand parent of this page.

This directive try to mimic the native position:sticky CSS attribute.

principle

<The app-stick> rely on a scroll event. The scrolling event is provided by the ScrollService. This ScrollService is (usually) injected in the app.component, & is fed by scrolling event of the main scrollable HTML element

The <appStick> consume this event, an then compute the relative position of the sticked element & its stick reference

Definition & usage

stick reference
(parent) element the sticked node should stick to. this reference node should have a css position set to "relative". Anyway, it is enforced automatically by the directive
stickable element
the element where the appStick directive is used. It should be a child element of the reference element.
appStick value
the stick reference node
stickable class
the app-stickable css class is set to element where appStick directive is defined.
scrolling
when scrolling enough, the css position of the sticked element is set to absolute (that' why the ref. element position must be relative)
sticked class
when the element become sticked, the app-sticked css class is added (& removed when not sticked anymore)
background
as the sticked element will overlap (when scrolling) other elements, it should have a background (so no be transparent)

Sample code

<div #myStyckRef> <!-- define the initial position to stick to -->
...
<h1 [appStick]=#myStyckRef>First section title<h1><!-- this element should stick when reaching the initial/reference position -->
...
<h1 [appStick]=#myStyckRef>Second section title<h1>
</div>
