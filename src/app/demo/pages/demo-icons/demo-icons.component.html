This demo illustrate incolib service & regisrry

Overview

Teh goal of the iconsService is to provide a easy way to add a large set of static icons (static: part of the GUI app bundle) & use them with as a collection (in addtion to be usable as regular mat-icon)

1st, the src/assets/iconlib.json should be generated at compilation time. See package.json build:iconlib. Basically, it just reflect the scanned directory folder tree.

Then, ar runtime / app startup, the incolib il loaded by the IconsService with a factory to ensure lib is loaded before any page is displayed

finaly, all the icons loaded from the iconlib are usable as regular angular icon, with namespace paramter

Column widths can be defin by css or by config

icon parameter:{{ icon }}
