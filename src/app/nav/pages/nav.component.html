This demo illustrate navigation components & behaviours

Overview

all GUI (display) components rely on loaded <module>-routing.module.ts. So have a look to tese modules, more specificaly to the data attribute of routes!

Routes

data.icon (optional)
define the icon to display on the right of the navigaiton item. This is a fontIcon name
data.animation (optional)
define the animation to apply when this route is activated
data.roles (optional)
define the (user) roles authorized to access the route
